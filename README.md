== Scalua ==

Scalua is a wrapper around [LuaJ 3.0](http://luaj.org/), written in Scala.

Scalua is currently a work-in-progress.

=== Features ===

* Operators (e.g. `+-*/^~`)
* Interoperability with LuaJ
* Pattern matching on `LuaValue`s
