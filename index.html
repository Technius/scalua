Scalua

Scalua is a wrapper for Luaj 3.0, written in Scala.

Requirements

  • Scala 2.11.x
  • Java 7 or higher

Build Configuration

To use Scalua as a library dependency, add the following to your build.sbt:

resolvers += Resolver.sonatypeRepo("snapshots")

libraryDependencies += "co.technius" %% "scalua" % "0.0.1-SNAPSHOT"

Please note that Scalua is currently still a work-in-progress and is thus only available as a snapshot.

Example Usage

import co.technius.scalua._

// Initialize globals
val g = Globals.standard

// Load code from a Source
val closure = g.load(scala.io.Source.fromFile("mycode.lua"), "mycode.lua")

// Call a function
closure.invoke()

// Construct numbers
val numbers = List(LuaInt(5), LuaDouble(10.3), LuaInt(7))

// Construct booleans and strings
val others = List(LuaBoolean(true), LuaString("Test"))

// Construct functions
val myFunction = LuaFunction {
  // Pattern match on Lua Values
  case LuaInt(5) :: List() => List(LuaString("Arg is"), LuaInt(5))
  case List() => List(LuaString("No arguments given"))
  case _ => List(LuaString("Incorrect usage"))
}

// Return only one argument
val secondFunction = LuaFunction.singleReturn {
  case List() => LuaString("No args")
  case _ => LuaString("Some args")
}

// Load code from Strings
g.load("a = 5").invoke()

// Get global variables
println(g("a")) // 5

// Set global variables
g("myFunction") = myFunction
g("secondFunction") = secondFunction

g.load("print('myFunction(): ' .. myFunction())", "test.lua").invoke()
g.load("print('secondFunction(123): ' .. secondFunction(123))", "test.lua").invoke()

// Call Lua functions from Scala
val moreCode = """
function myLuaFunction(x)
  print("x is " .. x)
  return x * 100
end
"""
g.load(moreCode).invoke()
// LuaValue extractors
val List(LuaInt(x)) = g("myLuaFunction").invoke(LuaInt(5)) // printed x is 500
println(x) // 500
